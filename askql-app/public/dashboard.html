How it works

From Data to Decision in 3 Steps

Spend less time wrangling data and more time acting on it.

1. Upload Your Data

Upload CSV, Excel, or JSON files. Your data is stored securely in an SQLite database for instant querying.

2. Ask in Plain English

No SQL knowledge needed. Tag your dataset with @ and ask questions in natural language. The AI generates and executes queries automatically.

3. Get Instant Insights

Receive AI-generated insights with automatic chart visualizations. Export conversations to PDF for easy sharing and documentation.

Two Powerful Modes

Ask Mode for safe data exploration and analysis. Agent Mode for data modifications with human-in-the-loop confirmation before any changes.

  • Ask Mode: Read-only queries with automatic visualizations and insights.
  • Agent Mode: Create, update, or delete records with required user confirmation.
